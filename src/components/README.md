# 🎬 Movie Recommendation System

A full-stack Machine Learning application that suggests movies based on content similarity using **K-Nearest Neighbors (KNN)**.

## 🚀 Features

- **FastAPI Backend:** Serves the ML model via a REST API.
- **Streamlit Frontend:** Interactive UI with movie posters from TMDB API.
- **Content-Based Filtering:** Uses `CountVectorizer` and `Cosine Similarity` on movie tags.
